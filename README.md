# algorithms-lib
Collection of widely used algorithms in competitive programming contests
Algorithms included are:

### Graph Algorithms:

  * SSSP algorithms: 
    * Dijkstra.

  * APSP Algorithms:
    * Floyd Warshall.
    
  * MST Algorithms:
    * Kruskal.

### Mathematics:
  * Grey Code.
  * Fibonacci O(log n).
  * Fast Power.
  * Binomial Coefficient.
  * Matrix Multiplication & Matrix Power.
  * Modular Inverse.
  
### Data Structures:
  * Sparse Table.
  * Disjoint Set (Union-Find).
  * Bitmasks.
  * Modified Stack & Queue - find min|max in O(1).
  
### Strings:
  * KMP.
  * Trie.

### Other:
  * Mo's Algorithm.
  * Maximum Sub-Array Sum(Kadane's Algorithm).
